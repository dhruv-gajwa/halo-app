## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User input (signup form) ‚Üí app ‚Üí `localStorage` | Plaintext passwords flow into `createVisitor`; they MUST be hashed before crossing into `localStorage`. |
| `localStorage` ‚Üí app | Disk values are attacker-controllable on shared machines; every read must be schema-validated. |
