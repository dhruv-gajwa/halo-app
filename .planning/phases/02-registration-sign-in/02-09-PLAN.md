- Primary use case (Select) — label `What will you use Halo for?`, placeholder `Pick what fits best`, options: Project management / Task tracking / Team coordination / Personal productivity / Just exploring
- Team size (NumberInput) — label `How many people on your team?`, placeholder `5`, min 1 max 10000

Wizard completion handler (the load-bearing part of this plan):

When step 4 submits successfully (Zod-validated):

1. **Defense in depth — re-validate every prior step**: `readWizardDraft()` then `step1Schema.parse(draft.step1)`, `step2Schema.parse(draft.step2)`, `step3Schema.parse(draft.step3)`. If any throws, redirect to `/signup` (the gate failed somewhere upstream; bail out cleanly rather than write half-data).

2. **Build a single object** combining all step values + the step-4 values just submitted.

3. **Call `authRepo.createVisitor(combined)`** — this hashes the password and writes the visitor record (Plan 02-03's contract).

4. **Call `authRepo.createWorkspace({ ownerVisitorId: visitor.id, companyName, companySize, industry, planTier })`** with values from `draft.step3`.

5. **Call `useAuthStore.getState().signInFromVisitor(visitor, workspace)`** — this writes `halo:v1:session` and updates the in-memory store atomically.

6. **Call `clearWizardDraft()`** — sessionStorage entry removed. The plaintext password in the draft is now gone too (mitigates T-02-17 retention window).

7. **`navigate('/app', { replace: true })`** — `replace: true` so the back button from `/app` doesn't go back to `/signup/preferences`.

Failure handling: any thrown error (localStorage quota, schema mismatch in defense-in-depth) sets a local React state `submitError: 'generic_failure'`, which renders the form-level `<Alert color="red" variant="light">Something went wrong — please try again.</Alert>` (Mantine Alert) above the form's Stack. Wizard draft is NOT cleared on failure — user can retry by clicking Create account again.

    Component:
    ```
    export function Step4PreferencesPage(): React.JSX.Element {
      const navigate = useNavigate()
      const draft = readWizardDraft()

      // Gate: all three prior steps must be complete. Any miss → redirect to /signup.
      if (!hasStep(draft, 'step1') || !hasStep(draft, 'step2') || !hasStep(draft, 'step3')) {
        return <Navigate to="/signup" replace />
      }

      const [submitError, setSubmitError] = useState<null | 'generic_failure'>(null)

      const form = useForm<Step4Values>({
        resolver: zodResolver(step4Schema),
        defaultValues: { primaryUseCase: undefined, teamSize: undefined, topGoals: [], ...(draft.step4 ?? {}) },
        mode: 'onSubmit',
      })

      const onSubmit = form.handleSubmit(async (step4Values) => {
        setSubmitError(null)
        try {
          // Defense in depth — re-read draft and re-validate every prior step before write.
          // If any prior-step Zod validation fails, fall back to /signup rather than write partial data.
          const freshDraft = readWizardDraft()
          const s1 = step1Schema.parse(freshDraft.step1)
          const s2 = step2Schema.parse(freshDraft.step2)
          const s3 = step3Schema.parse(freshDraft.step3)

          // Create the visitor — hashes password inside authRepo.createVisitor (Plan 02-03's contract).
          const visitor = await createVisitor({
            email: s1.email,
            password: s1.password,
            firstName: s1.firstName,
            lastName: s1.lastName,
            username: s1.username,
            jobTitle: s2.jobTitle,
            role: s2.role,
            yearsExperience: s2.yearsExperience,
            location: s2.location,
            primaryUseCase: step4Values.primaryUseCase,
            teamSize: step4Values.teamSize,
            topGoals: step4Values.topGoals,
          })

          // Create the workspace owned by the new visitor.
          const workspace = createWorkspace({
            ownerVisitorId: visitor.id,
            companyName: s3.companyName,
            companySize: s3.companySize,
            industry: s3.industry,
            planTier: s3.planTier,
          })

          // Sign the user in (writes session + updates in-memory store atomically per Plan 02-05).
          useAuthStore.getState().signInFromVisitor(visitor, workspace)

          // Clear the wizard draft — plaintext password retention window ends here (T-02-17 mitigation).
          clearWizardDraft()

          // Redirect into the authenticated area. replace:true so back button doesn't return to /signup/preferences.
          navigate('/app', { replace: true })
        } catch (err) {
          // Defense-in-depth Zod failure OR localStorage quota OR authRepo unexpected failure.
          // Either way, show the locked Alert copy and stay on /signup/preferences.
          // Wizard draft intentionally NOT cleared — user can retry.
          console.error('[signup] completion failed:', err)
          setSubmitError('generic_failure')
        }
      })

      const onBack = () => {
        writeWizardDraftStep('step4', form.getValues() as Partial<Step4Values>)
        navigate('/signup/company')
      }

      return (
        <Stack gap="md">
          <Title order={2}>Set up your workspace</Title>
          {submitError === 'generic_failure' && (
            <Alert icon={<IconAlertCircle size={18} />} color="red" variant="light">
              Something went wrong — please try again.
            </Alert>
          )}
          <form onSubmit={onSubmit} noValidate>
            <Stack gap="md">
              <Select
                label="What will you use Halo for?"
                placeholder="Pick what fits best"
                data={USE_CASE_OPTIONS as unknown as string[]}
                value={form.watch('primaryUseCase') ?? null}
                onChange={(value) => form.setValue('primaryUseCase', (value ?? '') as Step4Values['primaryUseCase'], { shouldValidate: false })}
                error={form.formState.errors.primaryUseCase?.message}
                pendoId={PENDO_IDS.signup.step4.useCase}
              />
              <NumberInput
                label="How many people on your team?"
                placeholder="5"
                min={1}
                max={10000}
                value={form.watch('teamSize') ?? ''}
                onChange={(value) => form.setValue('teamSize', typeof value === 'number' ? value : Number(value), { shouldValidate: false })}
                error={form.formState.errors.teamSize?.message}
                pendoId={PENDO_IDS.signup.step4.teamSize}
              />
              <MultiSelect
                label="What are you hoping to get out of Halo?"
                placeholder="Pick up to three"
                description="Select up to three."
                data={GOAL_OPTIONS as unknown as string[]}
                value={form.watch('topGoals') ?? []}
                onChange={(values) => form.setValue('topGoals', values as Step4Values['topGoals'], { shouldValidate: false })}
                error={form.formState.errors.topGoals?.message}
                maxValues={3}
                pendoId={PENDO_IDS.signup.step4.goals}
              />
              <Group justify="space-between" mt="xl">
                <Button type="button" variant="default" onClick={onBack} pendoId={PENDO_IDS.signup.step4.back}>
                  Back
                </Button>
                <Button type="submit" loading={form.formState.isSubmitting} pendoId={PENDO_IDS.signup.step4.submit}>
                  Create account
                </Button>
              </Group>
            </Stack>
          </form>
        </Stack>
      )
    }
    ```
